# 📊 IPL Analytics Dashboard (Power BI)

## 📌 Project Overview  
This project presents an interactive **IPL Analytics Dashboard** built using Power BI to analyze match, player, and toss data across multiple seasons. The objective is to uncover meaningful insights and support data-driven understanding of team performance, player contributions, and toss impact on match outcomes.

---

## 🎯 Objectives  
- Analyze IPL match trends across seasons and cities  
- Evaluate player performance (batting, bowling, fielding)  
- Understand the impact of toss decisions on match results  
- Identify top-performing teams under different conditions  

---

## 📊 Dashboard Features  

### 🔹 Match Overview  
- Total matches and distribution across cities  
- Season-wise trends in match count  
- Top teams based on wins  

### 🔹 Player Performance Analysis  
- KPIs: Total Runs, Wickets, Catches  
- Top run scorers, wicket takers, and fielders  
- Boundary analysis (4s & 6s)  

### 🔹 Toss Impact Analysis  
- Toss decision distribution (Bat vs Field)  
- Win % after winning toss  
- Match outcomes after winning and losing toss  
- Top teams performing with and without toss advantage  

---

## 🧠 Key Insights  
- Toss provides only a slight advantage (~50–55%) and is not decisive  
- Strong teams perform well regardless of toss outcome  
- Player performance significantly impacts match results  
- Boundaries and key bowling performances influence match outcomes  

---

## 🛠 Tools & Technologies  
- **Power BI** – Dashboard development & visualization  
- **DAX (Data Analysis Expressions)** – Measures and KPIs  
- **Power Query** – Data cleaning and transformation  

---

## 🚀 Conclusion  
This project demonstrates how data analytics can be applied to sports data to derive meaningful insights. It highlights that while toss has some influence, team strength and player performance are more critical in determining match outcomes.
